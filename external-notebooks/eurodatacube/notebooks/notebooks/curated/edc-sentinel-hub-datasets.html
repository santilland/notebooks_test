EDC Sentinel Hub - using xcube to access different data sets in Sentinel HubÂ¶

This notebook shows how to access different data sets through the Sentinel Hub API:

  1. Access Sentiel-2 L2A through xcube Sentinel hub Store
  2. Access Sentiel-2 L1C through xcube Sentinel hub Store
  3. Access Sentiel-1 GRD through xcube Sentinel hub Store
  4. Access DEM through xcube Sentinel hub Store
  5. Access to a custom data set ingested into Sentinel Hub via BYOD mechanism
from xcube_sh.config import CubeConfig
from xcube_sh.cube import open_cube
from xcube_sh.sentinelhub import SentinelHub
import xarray as xr
import numpy as np
x1 = 10.00  # degree
y1 = 54.27  # degree
x2 = 11.00  # degree
y2 = 54.60  # degree

bbox = x1, y1, x2, y2
spatial_res = 0.00018   # = 20.038 meters in degree
SH = SentinelHub()
SH.dataset_names
['LETML1', 'LMSSL1', 'LETML2', 'LTML1', 'LOTL2', 'S3OLCI', 'S2L2A', 'S3SLSTR', 'CUSTOM', 'LOTL1', 'LTML2', 'S2L1C', 'DEM', 'MODIS', 'HLS', 'S5PL2', 'S1GRD']

x1 = 1545577  # meters

bbox = x1, y1, x2, y2
width = 512
spatial_res = (x2 - x1) / width
height = max(1, round((y2 - y1) / spatial_res))
