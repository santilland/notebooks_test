This notebook demonstrates how the Eurocrops data can be used for analysis, using the geoDB.

from xcube_geodb.core.geodb import GeoDBClient
import time

geodb = GeoDBClient()
# Setting the database name; this database contains the Eurocrops data collections
db = 'geodb_b34bfae7-9265-4a3e-b921-06549d3c6035'

This step computes the area growing wheat in NRW (Germany), and measures the time needed to do that. Within the collection â€˜DE_NRW_2021_EC21â€™, the crop types are identified by different codes stored in the column â€˜codeâ€™; the values 112 and 115 identify wheat, therefore the respective query is 'or=(code.eq.112,code.eq.115)'. The sizes of the fields are stored in the column 'area_ha'.

tic = time.perf_counter()
df = geodb.get_collection('DE_NRW_2021_EC21', query='or=(code.eq.112,code.eq.115)', database=db)
